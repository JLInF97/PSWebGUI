# Hide-PSConsole

## SYNOPSIS
Hide the PowerShell console.

## SYNTAX

```powershell
Hide-PSConsole
```

## DESCRIPTION
Hide current PowerShell console window.
Sets ```$GLOBAL:CONSOLE_DISPLAY``` to 0

## EXAMPLES

### Example 1
```powershell
PS C:\> Hide-PSConsole
```
