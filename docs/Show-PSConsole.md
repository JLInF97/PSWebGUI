# Show-PSConsole

## SYNOPSIS
Shows the PowerShell console.

## SYNTAX

```powershell
Show-PSConsole
```

## DESCRIPTION
Shows the current hidden PowerShell console window.
Sets ```$GLOBAL:CONSOLE_DISPLAY``` to 1

## EXAMPLES

### Example 1
```powershell
PS C:\> Show-PSConsole
```
